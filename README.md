# Monkey Wordle

A Wordle-like game where you guess the monkey species from a blurred image.

## How to Play

- Guess the monkey species name
- Each incorrect guess reduces the blur slightly
- After 5 guesses, you'll get a hint
- After 6 guesses, you'll get another hint
- You have 7 guesses total to solve the puzzle

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to play.
